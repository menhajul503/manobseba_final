## Getting Started

### Login
1. Open the application in your browser
2. You'll see the login page with Manobseba branding
3. Enter your email address
4. Enter your password
5. Click "Sign in"

**First-time setup credentials:**
- Email: `admin@manobseba.com`
- Password: `password`

After login, you'll be redirected to the Dashboard.

## Dashboard

The Dashboard is your home screen showing an overview of the fund management system.

### Dashboard Components

#### 1. **Stats Cards** (Top Row)
Four key metrics displayed with icons:
- **Total Fund (৳)**: Sum of all donations minus all distributions
- **Total Members**: Number of active community members
- **Monthly Donation (৳)**: Total donations received this month
- **Total Distribution (৳)**: Total amount distributed this month

**Tip**: Click any stat card to view detailed information.

#### 2. **Income vs Expense Chart**
Visual representation of last 5 months' financial flow:
- **Green bars**: Income (donations)
- **Orange bars**: Expenses (distributions)
- Shows exact amounts for each month
- Helps identify spending patterns

#### 3. **Quick Actions**
Shortcuts to common tasks:
- 📝 **Record Donation** - Jump to donations page
- 👥 **Add Member** - Jump to members page
- 📢 **Create Notice** - Jump to notice board
- 📊 **New Distribution** - Jump to distributions page

#### 4. **Recent Activities Table**
Latest transactions and updates:
- Shows type: Donation, Distribution, Member, Notice
- Displays description and amount
- Shows date and time
- Color-coded status badges

**How to use**: Scroll through to see what happened recently. Click "View" to see more details.

## Members Management

Track and manage all community members.

### Accessing Members Page
1. Click **Members** in the sidebar
2. You'll see a list of all registered members

### Features

#### 1. **Add New Member**
```
1. Click "Add Member" button
2. Fill in member details:
   - Name (required)
   - Email (required)
   - Phone (optional)
   - Village (required)
   - Address (optional)
   - Join Date (required)
3. Click "Save"
```

#### 2. **Search Members**
```
1. Use the search box
2. Type member name or email
3. Results update in real-time
```

#### 3. **Filter by Status**
```
1. Click "Status" dropdown
2. Select: All, Active, or Inactive
3. Table updates automatically
```

#### 4. **Filter by Village**
```
1. Click "Village" dropdown
2. Select desired village
3. See only members from that village
```

#### 5. **View Member Details**
```
1. Find member in the table
2. Click "View" button
3. See full profile with:
   - Contact information
   - Total contributions
   - Donation history
   - Distribution history
```

#### 6. **Update Member**
```
1. Click "View" to open member
2. Click "Edit" button
3. Modify details as needed
4. Click "Save Changes"
```

#### 7. **Deactivate Member**
```
1. Open member profile
2. Change status to "Inactive"
3. Save changes
(Member can be reactivated later)
```

### Member Status
- **Active**: Member is current and eligible for distributions
- **Inactive**: Former member, archived for records

## Recording Donations

Track all fund contributions from members.

### Accessing Donations Page
1. Click **Donations** in the sidebar
2. Left side: Record new donation form
3. Right side: Donation history table

### Record a Donation

#### Step 1: Select Member
```
1. Click member name field
2. Start typing member name
3. Select from dropdown
```

#### Step 2: Enter Amount
```
1. Click Amount field (৳)
2. Enter donation amount
3. Example: 5000
```

#### Step 3: Select Payment Method
```
1. Click Payment Method dropdown
2. Choose from:
   - Cash
   - bKash
   - Nagad
   - Bank Transfer
```

#### Step 4: Set Date
```
1. Click Date field
2. Select donation date from calendar
3. Default is today's date
```

#### Step 5: Add Notes (Optional)
```
1. Type any additional notes
2. Example: "Monthly contribution"
```

#### Step 6: Submit
```
1. Click "Record Donation"
2. Success message appears
3. Receipt ID is auto-generated
4. Donation appears in history
```

### View Donation History

#### Features of History Table
- **Receipt ID**: Unique identifier for the donation
- **Donor**: Member who made the donation
- **Amount**: Donation amount in Taka (৳)
- **Method**: How payment was made
- **Date**: When donation was recorded

#### Filter Donations
```
1. Use Search box to find by:
   - Donor name
   - Receipt ID
2. Click "All Payment Methods" to filter by:
   - Cash only
   - bKash only
   - Nagad only
```

### Member Contribution Tracking
- Total contribution automatically updated
- Visible in member profile
- Updated in real-time

## Managing Distributions

Create and track distribution programs.

### Accessing Distributions Page
1. Click **Distributions** in the sidebar
2. See all distribution programs

### Create New Distribution Program

#### Step 1: Basic Information
```
1. Click "New Program"
2. Enter:
   - Program Name (e.g., "Eid Food Package")
   - Description (what is being distributed)
   - Total Budget (in Taka)
```

#### Step 2: Set Dates
```
1. Select Start Date
2. Select End Date (or leave for ongoing)
3. Click Create
```

### Manage Existing Program

#### View Program Details
```
1. Click any program card
2. Card expands to show:
   - Full description
   - Progress bar
   - Budget and spending
   - Number of recipients
   - Recent recipients list
```

#### Add Recipients
```
1. Click "View All Recipients" (inside expanded card)
2. Click "Add Recipient"
3. Select member
4. Enter amount to distribute
5. Add date
6. Click "Distribute"
```

#### Update Program Status
```
1. Click "Edit Program"
2. Change status:
   - Pending: Not yet started
   - In Progress: Currently distributing
   - Completed: All distributions done
3. Save changes
```

### Understanding Progress

#### Progress Bar
- **Green (75-100%)**: Most of budget allocated
- **Blue (50-74%)**: About half allocated
- **Orange (0-49%)**: Just started

#### Stats Displayed
```
Total Budget: Amount allocated for program
Spent: Amount already distributed
Recipients: Number of people helped
```

## Notice Board

Post and manage community announcements.

### Accessing Notice Board
1. Click **Notice Board** in the sidebar
2. Left side: Create notice form
3. Right side: Active and archived notices

### Post a New Notice

#### Step 1: Enter Title
```
1. Click Title field
2. Example: "Board Meeting Scheduled"
```

#### Step 2: Write Content
```
1. Click Content textarea
2. Write your announcement
3. Be clear and concise
```

#### Step 3: Set Priority
```
1. Click Priority dropdown
2. Select:
   - Low: General information
   - Medium: Important updates
   - High: Urgent notices
```

#### Step 4: Publish
```
1. Click "Post Notice"
2. Notice appears in feed immediately
3. All members can see it
```

### Managing Notices

#### View Active Notices
- Displayed in main feed
- Color-coded by priority:
  - **Red (High)**: Urgent
  - **Yellow (Medium)**: Important
  - **Green (Low)**: Information

#### Archive Notice
```
1. Click delete icon (trash)
2. Notice moves to archived section
3. Still visible to admins
```

#### View Archived Notices
- Shows previously active notices
- Displayed at bottom of page
- Kept for records

## Tips & Best Practices

### 💡 Dashboard Tips
- **Check regularly**: Review dashboard weekly for fund status
- **Monitor trends**: Watch Income vs Expense chart for spending patterns
- **Use quick actions**: Fastest way to perform common tasks

### 👥 Members Tips
- **Keep updated**: Update member status when they join/leave
- **Use villages**: Filter by village for targeted announcements
- **Track contributions**: Monitor member contributions over time

### 💰 Donations Tips
- **Receipt IDs**: Always verify receipt ID for records
- **Payment methods**: Track which methods are most used
- **Dates matter**: Record actual donation date, not just receipt date
- **Add notes**: Use notes field for extra context

### 📊 Distributions Tips
- **Plan budgets**: Allocate realistic budgets for programs
- **Track progress**: Monitor program progress regularly
- **Document recipients**: Keep detailed records of all distributions
- **Completion**: Mark programs completed when done

### 📢 Notice Board Tips
- **Use priorities**: Set correct priority level for urgency
- **Be clear**: Write notices that are easy to understand
- **Archives**: Check archives for past announcements
- **Frequency**: Don't overload with too many notices

## Common Tasks

### Task: Record Monthly Collection
```
1. Go to Members page
2. For each active member:
   - Go to Donations
   - Record their monthly donation
   - Select Cash if collected in person
   - Add date
3. Verify in Dashboard
```

### Task: Distribute Eid Funds
```
1. Go to Distributions
2. Create new program "Eid Distribution"
3. Set budget and dates
4. Add recipients with amounts
5. Record as "In Progress"
6. Update status to "Completed" when done
```

### Task: Announce Important Meeting
```
1. Go to Notice Board
2. Click "Post Notice"
3. Title: "Board Meeting on [Date]"
4. Set priority to High
5. Include time and location
6. Post
```

### Task: Generate Fund Report
```
1. Go to Dashboard
2. Note Total Fund amount
3. Use statistics shown
4. Export screenshot or note values
5. Share with board
```

## Troubleshooting

### Issue: Can't login
- **Solution**: Check email and password are correct
- **Reset**: Use "Forgot Password" or contact admin

### Issue: Donation not appearing
- **Solution**: Refresh page (F5)
- **Check**: Verify you selected correct member

### Issue: Can't find member
- **Solution**: Clear filters and search again
- **Check**: Verify member name spelling

### Issue: Chart not loading
- **Solution**: Wait a moment for data to load
- **Refresh**: Close and reopen page

## Frequently Asked Questions

**Q: Can I edit a recorded donation?**
A: Yes, open donation details and click "Edit"

**Q: How is Total Fund calculated?**
A: Total Fund = All Donations - All Distributions

**Q: Can I delete a member?**
A: Instead of deleting, set status to "Inactive"

**Q: What happens if I exceed distribution budget?**
A: System allows it, but flags as over budget

**Q: Can members view their own donations?**
A: Yes, if they login to the system

**Q: How long are notices kept?**
A: Until manually deleted or set expiration date
